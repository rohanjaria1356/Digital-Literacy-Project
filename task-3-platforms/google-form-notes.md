## How to Create This Form
1. Go to [forms.google.com](https://forms.google.com)
2. Click **Blank form** (+ icon)
3. Set the title to "Digital Literacy Awareness Quiz"
4. Add each question above, selecting the correct question type (Multiple Choice or Short Answer)
5. For MCQ questions, mark the correct answer and enable "Make this a quiz" under Settings
6. Click **Send** → copy the shareable link
7. View responses in the linked **Google Sheet** (Responses tab → green Sheets icon)
