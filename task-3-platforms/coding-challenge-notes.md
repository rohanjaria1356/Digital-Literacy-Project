# Task 3 — Part A: Coding Challenge Notes

## Platform Chosen: HackerRank (hackerrank.com)

### About HackerRank
HackerRank is a competitive programming and technical skill assessment platform widely used by companies for hiring and by students for practice. It offers challenges across domains including algorithms, data structures, Python, Java, SQL, and more.

### Challenge Completed
- **Challenge Name:** Solve Me First
- **Domain:** Algorithms → Warmup
- **Difficulty:** Easy
- **Language Used:** Python

### Challenge Description
The "Solve Me First" challenge is an introductory problem where you are given two integers and must return their sum. While simple, it teaches the fundamentals of reading input, processing data, and producing output in the HackerRank environment.

### Solution Approach
```python
def solveMeFirst(a, b):
    return a + b

num1 = int(input())
num2 = int(input())
res = solveMeFirst(num1, num2)
print(res)
```

### What I Learned
This challenge helped me understand how online judges work — how input is read from standard input, processed through a function, and the output is validated against expected results. It gave me confidence to attempt more complex problems next.
