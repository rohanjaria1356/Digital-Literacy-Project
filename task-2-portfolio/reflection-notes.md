# Task 2 — Student Digital Portfolio: Reflection Notes

## Platforms Chosen

For my digital portfolio, I set up professional profiles on three platforms that are most relevant to my journey as a Computer Science student:

### 1. GitHub (github.com)
GitHub is the world's leading platform for version control and collaborative software development. As a CSE student, it serves as both a code repository and a living portfolio of my programming projects. I created a profile README showcasing my name, branch, year, and learning goals. Over the next four years, I plan to use GitHub to host course projects, contribute to open-source repositories, and build a track record that recruiters and graduate schools can review.

### 2. LinkedIn (linkedin.com)
LinkedIn is the primary professional networking platform used globally for career development. I set up my profile with my education details — B.Tech in Computer Science & Engineering at VIT Bhopal University, with an expected graduation in 2029. I plan to use LinkedIn to connect with alumni, follow industry leaders, discover internship opportunities, and document my academic and professional milestones.

### 3. Stack Overflow (stackoverflow.com)
Stack Overflow is the world's largest community-driven Q&A platform for programmers and developers. I chose Stack Overflow because it is where real-world coding problems are solved every day — reading and understanding answers here builds debugging skills faster than any textbook. As a first-year student, I plan to start by searching for solutions to problems I encounter in my coursework, gradually move to answering beginner-level questions, and build a reputation score that reflects my growing technical knowledge.

## Future Plans
Over the next four years, I aim to actively maintain all three profiles — pushing code to GitHub regularly, expanding my LinkedIn network with meaningful connections, and building my Stack Overflow reputation by contributing quality answers. These platforms together will form the backbone of my professional digital identity as I progress through my degree and prepare for industry.
